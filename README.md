# CW Decoder for UIAPduino 

UIAP Pro Microで動作するサンプルプログラムです。  
ビルド環境はVSCode + ch32v003funが必要です。  
書き込は、WCH-LinkEまたは、UIAPのUSB端子から行います。

## UIAPsideToneDDS
モールス信号のON/OFF信号を入力し、700Hzのサイドトーンを出力します。
歪の少ないサイン波を出力します。
